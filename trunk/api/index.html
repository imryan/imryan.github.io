StudentWatch REST API

The official documentation of the StudentWatch REST API.

Business Service

POST /BusinessService/GetBusinessForUser

Returns the business that the user is assigned to.

Params:
    - ID

POST /BusinessService/SaveBusiness

Saves or updates a Business account.

Params:
    - ID (if updating)
    - Name
    - DeviceLimit
    - EventsLimit
    - StudentLimit
    - UserLimit

Events Service

POST /EventsService/SaveEvent

Saves or updates an Event,

Params:
    - ID (if updating)
    - BusinessID
    - City
    - State
    - Date_Time
    - Description
    - DriverID
    - Location

POST /EventsService/SaveEventForStudent

Assigns a student to an event.

Params:
    - StudentID
    - EventID

POST /EventsService/GetEventsByBusiness

Returns list of events for the given Business.

Params:
    - BusinessID

POST /EventsService/GetEventsByRegistrationCode

Returns list of events for the given registration code.

Params:
    - RegistrationCode

GET /EventsService/GetEventsForStudent/[StudentID]

Returns list of events for the given Student.

Params:
    - StudentID

POST /EventsService/DeleteEvent

Deletes an Event.

Params:
    - EventID

POST /EventsService/DeleteEventForStudent

Unassigns an Event for a Student.

Params:
    - StudentID
    - EventID

User Service

POST /UserService/ActivateUser

Activates a User account.

Params:
    - ID

POST /UserService/DeactiveUser

Deactivates a User account.

Params:
    - ID

POST /UserService/ValidateUser

Validates a User account.

Params:
    - ID

POST /UserService/SaveUser

Saves or updates a User account.

Params:
    - ID (if updating)
    - BusinessID
    - Email
    - FirstName
    - MiddleName
    - LastName
    - Password
    - Role
    - Username

POST /UserService/DeleteUser

Deletes a User account.

Params:
    - ID

POST /UserService/GetUserInfoById

Returns information about the User given.

Params:
    - ID

POST /UserService/GetUserListByBusiness

Returns a list of Users for the Business.

Params:
    - BusinessID

Student Service

POST /StudentService/AddStudent

Adds a new Student into the database.

Params:
    - BusinessID
    - CellNumber
    - DOB
    - ECName
    - ECNumber
    - EventID
    - FirstName
    - MiddleName
    - LastName
    - Notes
    - Picture
    - SchoolID

GET /StudentService/DeleteStudent/[StudentID]

Deletes a Student from the database.

Params:
    - StudentID

POST /StudentService/EditStudent

Edits a Student

Params:
    - ID
    - BusinessID
    - CellNumber
    - DOB
    - ECName
    - ECNumber
    - EventID
    - FirstName
    - MiddleName
    - LastName
    - Notes
    - Picture
    - SchoolID

GET /StudentService/GetLastScanLocation/[StudentID]

Returns the ID of the Stop where the student was last scanned.

Params:
    - StudentID

GET /StudentService/GetReportForStudent/[StudentID]

Returns a report of all the places the Student was scanned at.

Params:
    - StudentID

GET /StudentService/GetStudentById/[StudentID]

Returns a Student object with information for the Student given.

Params:
    - StudentID

GET /StudentService/GetStudentByNfc/[NfcCode]

Returns a Student by their NFC code.

Params:
    - NfcCode

GET /StudentService/GetStudentByQr/[QrCode]

Returns a Student by their QR code.

Params:
    - QrCode

GET /StudentService/GetStudentsForEvent/[BusinessID]/[EventID]

Returns a Student by their QR code.

Params:
    - BusinessID
    - EventID

POST /StudentService/SaveReport

Saves a new Stop to a StudentReport object.

Params:
    - BusinessID
    - EventID
    - Latitude
    - Longitude
    - Timestamp
    - SchoolID
    - StopID
    - StudentID

Stop Service

POST /StopsService/SaveStop

Saves a new Stop to the database.

Params:
    - BusinessID
    - EventID
    - Description
    - Location

GET /StopsService/GetStopById/[BusinessID]/[EventID]/[ID]

Returns a Stop object by its ID.

Params:
    - BusinessID
    - EventID
    - ID

GET /StopsService/GetStopsForEvent/[BusinessID]/[EventID]

Returns a list of Stops for the Event.

Params:
    - BusinessID
    - EventID

POST /StopsService/GetStopById/SetStopCompleted

Sets a Stop as completed.

Params:
    - ID

POST /StopsService/DeleteStop

Deletes a Stop.

Params:
    - ID

GET /SchoolService/GetSchoolsForBusiness/[BusinessID]

POST /SchoolService/DeleteSchool

Deletes a new School to the database.

Params:
    - ID
